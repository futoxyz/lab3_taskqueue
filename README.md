# Задачи
`src\task.py`
Task представляется через обычный класс. Любая задача имеет:
- **id** — уникальный идентификатор задачи.
- **description** — описание задачи.
- **priority** — порядок приоритета задачи. По умолчанию принимает *0*.
- **status** — текущий статус задачи. По умолчанию принимает *pending*.

**id**, **description** и **priority** проверяются через дескрипторы в том же файле.

Для **status** проверка происходит через сеттер внутри класса.

Допустимые значения статуса: *pending*, *in_progress*, *failed*, *completed*.
